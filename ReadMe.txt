Readme angående GitInspector kod.

Vi har haft problem undertiden vi refaktorerat vår kod. När man har ändrat i vissa packages har vi tagit över de andras kod enligt gitInspector. Detta leder till att
vi har olika mängder kod och det kan skilja väldigt mycket. Vi har i stort sett programmerat lika stor del av projektet och ville framföra detta så det inte
blir någon som får lida för att refaktoreringen har tagit över deras kod.
